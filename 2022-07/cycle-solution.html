We didn't use the last string ("AMSTERDAMSIXTYSIX") and one graph yet. We got a new phrase "CYCLE LF ZUIDERZEEROUTE". LF-routes are long-distance cycling routes in Netherlands, and LF Zuiderzeeroute is a route that goes around Zuiderzee in a cycle. LF Zuiderzeeroute goes through 9 of the cities/towns used in the puzzle so far, which corresponds to the 9 nodes in the unused graph. Using the string, node 66 can be matched with Amsterdam, and by guessing (sorry) the right orientation (node 83 corresponds with going clockwise to Monnickendam) all nodes can be matched to a city/town. Using tens digit to sort and ones digit to index, we can extract the final answer HOVENRING, which is a cyclic bridge for cyclists in Netherlands.
